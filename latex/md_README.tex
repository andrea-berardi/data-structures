\hypertarget{md_README_autotoc_md5}{}\doxysection{Notes}\label{md_README_autotoc_md5}
Unless I\textquotesingle{}m missing something, the project should be 100\% free from\+:


\begin{DoxyItemize}
\item \+:cyclone\+: Undefined Behaviors (checked with Clang\textquotesingle{}s sanitizers)
\item \+:bug\+: Bugs ({\itshape hopefully})
\item \+:sweat\+\_\+drops\+: Memory Leaks (checks made with Clang and Valgrind)
\end{DoxyItemize}

The {\ttfamily CMake\+Lists.\+txt} (build file) ensures that debug builds have runtime sanity checks (provided by Clang) and the absence of warnings$\ast$°$^\wedge$ hints that there aren\textquotesingle{}t weird things going on. In addition, Cppcheck (a static C/\+C++ code analyzer) and Valgrind confirmed that everything is compliant and safe. The {\ttfamily DEBUG} flag on the code even allows performing additional correctness checks (it ensures the data structures work correctly by using many antagonist functions). I\textquotesingle{}m keeping it as a {\ttfamily const} flag to allow the compiler to better optimize the code when it is set to {\ttfamily false} (hopefully it recognizes that its value never changes and removes the {\ttfamily if} branch of the checks completely from the binary).

I\textquotesingle{}m planning to add more specific directions on how to run the project, but afaik it should be already out-\/of-\/the-\/box.

I use CLion as my IDE and Clang as my compiler. The project manager is CMake.

I tried to keep the algorithms as close as possible to those shown on the book {\itshape Introduction to Algorithms}, but it wasn\textquotesingle{}t always possible. Nevertheless, I have {\bfseries{not}} used any additional resources apart my professor\textquotesingle{}s PDFs and my tutor\textquotesingle{}s laboratory documentations.

\+\_\+$\ast$\+: Unfortunately there\textquotesingle{}s a tiny warning due to my use of {\ttfamily rand()} in the {\ttfamily gen\+\_\+rand\+\_\+array()} function. Basically, the IDE is saying that it has limited randomness, which is fine and totally expected. Please ignore.\+\_\+

{\itshape °\+: The warnings about recursive calls can be ignored too, they\textquotesingle{}re necessary to make some algorithms work properly.}
